Analysis of Chronic Kidney Disease using Machine Learning Techniques

  • Tech Stack: Python, Machine Learning
  • Tools: Google Colabaratory, Numpy, MatPlotlib
  • Project: Link
  • Project paper: paper report

Developed a machine learning model to detect CKD at an early stage when symptoms may be subtle.

The model was trained on real time dataset. The Analysis done with three training and testing ratios 67:33, 70:30 and 80:20. We had done analysis of KNN algorithm by taking k values from 1 to 5.

We had done analysis of our model by using 3 different training testing ratios and with 3 different algorithms. By comparing different ratios we got better accuracy for 70:30 ratio. We got better accuracy for Decision Tree algorithm compared to KNN and Naive Bayes algorithms.
